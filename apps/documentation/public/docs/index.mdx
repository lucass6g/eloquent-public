### O que é o Eloquent UI?
O EloquentUI é um conjunto de componentes React que foram projetados para serem reutilizados em todos os projetos Elo. O objetivo é simplificar o processo de construção de interfaces de usuário bonitas, funcionais e acessíveis.

### Contribuição
Você pode contribuir com o EloquentUI escrevendo códigos na [Azure](https://dev.azure.com/elobr/cross/_git/eloquent), mantendo nossa documentação no [Confluence](https://elo-confluence.atlassian.net/wiki/spaces/Frontend/pages/2176418675/Eloquent+-+Design+System+Elo) sempre atualizada e também reportando bugs.
